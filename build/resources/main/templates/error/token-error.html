토큰 인증 오류

죄송합니다. 토큰 인증에 실패했습니다.

업무 화면에 접근하기 위해서는 유효한 토큰이 필요합니다.

발생 가능한 원인:

  • 토큰이 만료되었습니다 (24시간 경과)
  • 토큰이 유효하지 않습니다
  • 토큰이 제공되지 않았습니다
  • 토큰이 비활성화되었습니다
  • 잘못된 토큰 형식입니다

해결 방법:

아래 "다시 로그인" 버튼을 클릭하여 새로운 토큰을 발급받으세요.

로그인 후 생성된 토큰을 HTTP 헤더에 다음과 같이 추가하세요:

Authorization: Bearer [토큰값]
